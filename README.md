# 🎶 Playlist

## 🚀 Getting started

Clone o projeto e acesse a pasta Playlist.

```bash
git clone https://github.com/AOBarbosa/Playlist.git

cd Playlist

```

Para rodar o projeto, digite os seguintes comandos:

```bash
cd build

make

./program
```
